Use `bun` instead of `npx` or `npm` for running JavaScript/Node tooling commands.

**Why:** User preference — they use bun as their JS package manager and runtime.

**How to apply:** Always use `bun run`, `bunx`, or `bun` instead of `npx`, `npm run`, etc.
